・はじめに

	ダウンロード頂きありがとうございます。

	このプログラムは簡単にサウンドノベルフラッシュを作成できるツールです。
	data.xmlを書き換えるだけで、すぐにサウンドノベルフラッシュを
	作成する事ができます。
	
	是非素晴らしい作品を作ってください！
	作った作品は教えていただけると嬉しいです〜♪
	掲示板にお気軽に書き込んでくださいませ。

	池田プロダクション
	http://ikeda.gotdns.com/
	
	石井高雅

・ファイル構成

	同梱されている、「SNFS簡易仕様書.txt」を御覧下さい。

・インストール方法

	解凍して頂ければ、使用できます。
	SNFSを使って制作したフラッシュの公開方法ですが、
	通常のフラッシュ公開方法と特に変化はありませんが、
	以下のファイルはアップロード必須になります。

	data.xml
	init.xml
	bgmフォルダ
	imagesフォルダ
	.swf（フラッシュファイル）
	　（SoundNovelXML.swf等、フラッシュの名前は変更してもかまいません）

・アンインストール方法

	ファイルを全て削除して下さい。

・使い方

	docsフォルダのSNFS文法説明書を御覧になって頂き、
	data.xmlの作成を行ってください。
	
・生成されるフラッシュの見方

	左クリックを押すと一文字づつ３行まで文字が表示されます。
	また、スペースを押すと３行を一度に表示します。

	３行の表示がおわる段階で、次のXMLタグを読み込み表示を行います。
